## VGG Image Annotator （VIA）图像标注工具用户指南


### VIA 简介

VGG Image Annotator (VIA)是一款开源的图像标注工具，由牛津大学计算机视觉组 Visual Geometry Group开发。可以在线和离线使用，可标注矩形、圆、椭圆、多边形、点和线。标注完成后，可以导出为csv和json文件格式。
